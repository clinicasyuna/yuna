# Configuração de Domínio Personalizado

## Configuração DNS Recomendada

### Para o domínio: clinicasyuna.com.br

```
Tipo: CNAME
Nome: app
Valor: clinicasyuna.github.io
TTL: 3600
```

### Resultado final: 
- **Portal Admin**: https://app.clinicasyuna.com.br/admin/
- **Portal Acompanhantes**: https://app.clinicasyuna.com.br/acompanhantes/

## Configuração no GitHub

1. Acesse: Settings → Pages
2. Custom domain: `app.clinicasyuna.com.br`
3. Enforce HTTPS: ✅ Habilitado

## Vantagens do Domínio Personalizado

- ✅ **Profissionalismo**: URL própria da clínica
- ✅ **Segurança**: HTTPS automático
- ✅ **Branding**: Marca YUNA mais forte
- ✅ **Confiança**: Usuários confiam mais
- ✅ **SEO**: Melhor indexação
